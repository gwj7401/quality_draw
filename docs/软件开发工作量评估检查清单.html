📊 过程监督检查项

代码提交记录检查

  • 获取Git仓库访问权限
  • 检查提交频率是否合理
  • 检查提交时间分布
  • 检查代码变更量统计

常用Git命令：

# 查看时间段内的提交统计
git log --since="开始日期" --until="结束日期" --oneline --shortstat

# 查看每人提交量
git shortlog -sn --since="开始日期"

# 查看详细变更
git log --stat --since="开始日期"

# 查看某人的提交
git log --author="开发者名" --oneline
