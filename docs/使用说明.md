# 宁夏特检院质量监督检查抽签程序

## 使用说明

### 程序功能

本程序用于宁夏特检院质量监督检查的抽签工作，确保监督检查的公正性和随机性。

### 主要功能

1. **部门选择**：左侧面板可选择被检查的部门
2. **随机抽签**：点击开始后，人名快速滚动，点击停止后减速并显示最终结果
3. **同部门规避**：自动排除被检查部门的人员
4. **连续规避**：同一人不会连续两次被抽中
5. **双人抽取**：综合类部门（如石嘴山分院）同时抽取承压和机电两类专责
6. **结果导出**：支持导出到Excel文件
7. **打印功能**：支持浏览器打印

### 部门分类

| 类型 | 部门名称 | 需要抽取 |
|-----|---------|---------|
| 综合类 | 宁东分院、石嘴山分院、吴忠分院、中卫分院、固原分院 | 承压 + 机电 |
| 承压类 | 承压特种设备一部、承压特种设备二部、综合检验检测站 | 仅承压 |
| 机电类 | 机电特种设备一部、机电特种设备二部 | 仅机电 |

### 操作步骤

1. **添加质量专责**
   - 点击底部"数据管理"按钮
   - 在"质量专责管理"标签页添加人员
   - 选择姓名、所属部门、专业类型

2. **进行抽签**
   - 在左侧选择被检查部门
   - 点击"开始抽签"按钮
   - 观察滚动动画
   - 点击"停止"按钮
   - 查看抽签结果

3. **导出结果**
   - 点击"导出Excel"按钮
   - 文件自动保存到桌面

4. **打印结果**
   - 点击"打印"按钮
   - 在浏览器中进行打印

### 数据存储

程序数据存储在exe同级的`data`目录下：
- `departments.json` - 部门数据
- `specialists.json` - 质量专责数据
- `records.json` - 抽签记录

### 注意事项

1. 首次使用请先添加质量专责人员
2. 确保每个部门都有足够的质量专责人员
3. 抽签记录会自动保存，可随时查看历史
